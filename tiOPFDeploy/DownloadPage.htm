Version: tiOPF2

tiOPF2 - The latest work-in-progress featuring many class, method and unit name changes. The unit and class names make this version more intuitive than previous versions of tiOPF. tiOPF2 contains basically the same functionality as the build below, but with the new class and unit names. tiOPF2 is not regularly unit tested yet, so if you need stability, this is not the release for you. You can download a zip file containing tiOPF2 source here. (The web server on the build box is built with IndyHTTPServer, and I have set the MIME type incorrectly for zip file. This causes problems when downloading with Mozilla Firefox, so please use IE to download this file.)

tiOPF Support apps Applications to help use the tiOPF including:
  • tiIBSQLMonitor - watch communication between yor application and the Interbase or Firebird server.
  • tiSQLEditor - Editor for SQL with colour syntax hilighting. Will connect to any tiOPF SQL persistence layer.
  • tiSQLManager - lets you store an application's SQL in the database. Great for complex queries where a DBA may be required to tune the SQL.
  • tiDBParamMgr - edit database connection parameters that are stored in an encrypted file.
