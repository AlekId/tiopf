How to use live templates
=========================
To use these "live templates," all you need to do is copy the XML snippet provided and save it in 

  "C:\Program Files (x86)\Embarcadero\RAD Studio\10.0\ObjRepos\en\Code_Templates\Delphi" 

with an xml extension.

Alternate (preferred) place to put them are in your local user directory:

  c:\Users\<username>\Documents\RAD Studio\code_templates\
